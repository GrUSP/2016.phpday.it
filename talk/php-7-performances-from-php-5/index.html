PHP 7 performances from PHP 5

Date:
Track:
Language:
english

Abstract

PHP 7 is incredibly faster than PHP 5. In this talk, we will compare profiling of applications against PHP 7 and PHP 5, and point some scenarios where PHP 7 really outperforms PHP 5. We’ll then analyze together the reasons of such performance boosts by explaining the low level tricks that have been brought to PHP 7. New memory manager, better aligned structures and lots of CPU-efficiency tricks are in, we’ll explore them all.
